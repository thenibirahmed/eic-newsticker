EIC (Elegance In Code) Newsticker is a plugin to display a news ticker on your website.

== Description ==

EIC Newsticker displays the latest 5 posts on your WordPress website in a scrolling news ticker format. Customize the appearance of the ticker using shortcode attributes.

Features include:
- Display latest 5 posts
- Customizable title and content styles
- Easy shortcode integration

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `eic-newsticker` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcode `[eic_newsticker]` to display the news ticker on your website
